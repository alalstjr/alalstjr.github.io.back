# 포워드(forward) 방식

RequestDispatcher rd = request.getRequestDispatcher(url);
rd.forward(request, response);

혹은

<!-- jsp 액션 태그를 이용한 forward -->
<jsp:forward page="sessionResult.jsp"></jsp:forward>

주소는 그대로, 화면은 변경
대량의 데이터 전달이 가능합니다.

# Redirect 방식

response.sendRedirect(url);

주소 변경, 화면도 변경
파라미터 스트링만 전달 가능합니다.
